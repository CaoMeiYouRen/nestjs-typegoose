Schema Options

To add custom mongoose schema options you can simply change Typegoose.forFeature to the following format:

@Module({
imports: [
TypegooseModule.forFeature([
{
typegooseClass: Cat,
schemaOptions: {
collection: "ADifferentCollectionNameForCats"
}
}
])
]
})
export class CatsModule {}
